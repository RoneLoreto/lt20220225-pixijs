# About

react-pixi を試してみたリポジトリです

## 起動手順

`docker compose up -d`

`docker compose exec react-pixi ash`

`yarn start`

## 踏んだ手順メモ

`yarn create react-app react-pixi --template typescript`

`yarn add --dev pixi.js @inlet/react-pixi`
